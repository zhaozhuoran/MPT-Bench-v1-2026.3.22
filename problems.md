### Part 4

```latex

18.
(1).
Let \(\{a_n\}\) and \(\{b_n\}\) be given by \(a_1=1,\ a_2=2,\ a_3=3,\ b_1=1,\ b_2=3,\ b_3=5\). Define
\[
c_n=\max\{\,b_1-a_1n,\ b_2-a_2n,\ \dots,\ b_n-a_nn\,\}.
\]

For \(n=1\):
\[
c_1=\max\{b_1-a_1\}=\max\{0\}=0.
\]

For \(n=2\):
\[
c_2=\max\{b_1-2a_1,\ b_2-2a_2\}=\max\{-1,\ -1\}=-1.
\]

For \(n=3\):
\[
c_3=\max\{b_1-3a_1,\ b_2-3a_2,\ b_3-3a_3\}=\max\{-2,\ -3,\ -4\}=-2.
\]

We now prove that for every \(n\in\mathbf{N}^*\) with \(n\ge 2\),
\[
c_n=b_1-a_1n.
\]
For \(n\in\mathbf{N}^*\) and \(2\le k\le n\),
\[
(b_k-a_kn)-(b_1-a_1n)=(k-1)(2-n).
\]
Since \(k-1>0\) and \(2-n\le 0\), we have \((b_k-a_kn)-(b_1-a_1n)\le 0\), hence \(b_1-a_1n\ge b_k-a_kn\). Therefore for all \(n\ge 2\),
\[
c_n=b_1-a_1n=1-n.
\]
Thus
\[
c_{n+1}-c_n=-1
\]
and also \(c_2-c_1=-1-0=-1\). Hence \(c_{n+1}-c_n=-1\) holds for all \(n\in\mathbf{N}^*\), so the sequence \(\{c_n\}\) is an arithmetic progression.

(2).
Let the common differences of \(\{a_n\}\) and \(\{b_n\}\) be \(d_1\) and \(d_2\) respectively. Consider the values
\[
b_1-a_1n,\ b_2-a_2n,\ \dots,\ b_n-a_nn.
\]
For any \(i\) with \(1\le i\le n\),
\[
b_i-a_in=\bigl[b_1+(i-1)d_2\bigr]-\bigl[a_1+(i-1)d_1\bigr]\cdot n
=(b_1-a_1n)+(i-1)(d_2-d_1n).
\]
We discuss three cases for \(d_1\).

**Case 1: \(d_1=0\).**
Then
\[
b_i-a_in=(b_1-a_1n)+(i-1)d_2.
\]
If \(d_2\le 0\), then \((b_i-a_in)-(b_1-a_1n)=(i-1)d_2\le 0\), so for each fixed positive integer \(n\),
\[
c_n=b_1-a_1n,
\]
and \(c_{n+1}-c_n=-a_1\). Thus \(\{c_n\}\) is an arithmetic sequence.

If \(d_2>0\), then
\[
(b_i-a_in)-(b_n-a_nn)=(i-n)d_2\le 0,
\]
so for each fixed \(n\),
\[
c_n=b_n-a_nn=b_n-a_1n,
\]
and \(c_{n+1}-c_n=d_2-a_1\). Hence \(\{c_n\}\) is an arithmetic sequence. In this case take \(m=1\); the sequence \(\{c_n\}\) is arithmetic from the first term.

**Case 2: \(d_1>0\).**
Then \(-d_1n+d_2\) is an affine function of \(n\) with negative slope. Thus there exists \(m\in\mathbf{N}^*\) such that for all \(n\ge m\), \(-d_1n+d_2<0\). For such \(n\),
\[
(b_i-a_in)-(b_1-a_1n)=(i-1)(-d_1n+d_2)\le 0\quad(1\le i\le n),
\]
so \(c_n=b_1-a_1n\) for all \(n\ge m\). Then \(c_{n+1}-c_n=-a_1\), and \(\{c_n\}\) is an arithmetic sequence from the \(m\)-th term onward.

**Case 3: \(d_1<0\).**
Now \(-d_1n+d_2\) is an affine function of \(n\) with positive slope. Hence there exists \(s\in\mathbf{N}^*\) such that for all \(n\ge s\), \(-d_1n+d_2>0\). For such \(n\),
\[
(b_i-a_in)-(b_n-a_nn)=(i-n)(-d_1n+d_2)\le 0\quad(1\le i\le n),
\]
so \(c_n=b_n-a_nn\) for all \(n\ge s\). Then
\[
\frac{c_n}{n}=\frac{b_n-a_nn}{n}=-a_n+\frac{b_n}{n}.
\]
Using arithmetic progression formulas,
\[
\frac{c_n}{n}=-d_1n+(d_1-a_1+d_2)+\frac{b_1-d_2}{n}.
\]
Set \(A=-d_1>0,\ B=d_1-a_1+d_2,\ C=b_1-d_2\). Then
\[
\frac{c_n}{n}=An+B+\frac{C}{n}.
\]
We show that for any positive number \(M\) there exists \(m\) such that for all \(n\ge m\), \(\dfrac{c_n}{n}>M\).

- If \(C\ge 0\), choose \(m=\bigl\lfloor\frac{|M-B|}{A}\bigr\rfloor+1\). For \(n\ge m\),
\[
\frac{c_n}{n}\ge An+B\ge A\Bigl\lfloor\frac{|M-B|}{A}\Bigr\rfloor+1+B> M.
\]

- If \(C<0\), choose \(m=\bigl\lfloor\frac{|M-C-B|}{A}\bigr\rfloor+1\). For \(n\ge m\),
\[
\frac{c_n}{n}=An+B+\frac{C}{n}\ge Am+B+C>A\Bigl\lfloor\frac{|M-C-B|}{A}\Bigr\rfloor+B+C\ge M.
\]

Thus for any \(M>0\) there exists \(m\) such that for all \(n\ge m\), \(\dfrac{c_n}{n}>M\).

Combining the three cases, the proposition is proved.
```
