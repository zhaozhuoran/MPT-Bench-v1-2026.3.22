# MPT-Bench-v1 (2026.3.22)

这是一个用于评估大型语言模型（LLMs）解决高中/高考级别复杂数学问题能力的评测基准。

[在 Kaggle Benchmarks 上查看](https://www.kaggle.com/benchmarks/yearcakes/mpt-bench-2026-3-22)

## 简介

本仓库包含了一套数学题以及基于 `kaggle_benchmarks` 框架的自动化测评脚本。

**试题来源**：北京市海淀区某 top3 学校的高二某次考试试题。

试卷满分为 100 分，共分为 4 个部分（18道题）：

- **Part 1** (`task-1.py`): 10 道单项选择题（30分，每题3分）。
- **Part 2** (`task-2.py`): 5 道填空/选择/区间题（30分，每题6分）。
- **Part 3** (`task-3.py`): 2 道证明与计算大题（25分，分别为10分和15分）。
- **Part 4** (`task-4.py`): 1 道综合压轴证明题（15分）。

## 评分机制（Evaluation Scheme）

- **客观题（Part 1 & 2）**：采用精准匹配或正则表达式等去校验 LLM 根据预期 Schema 输出的 JSON 结果。**只有答案完全正确才给分**，答案格式错误、结果错误或不全面均不得分。
- **主观证明题（Part 3 & 4）**：使用裁判大模型（`judge_llm`）对推理过程进行综合打分。在此环节，**允许模型使用不同的有效证明方法**；如果证明过程中断或出现逻辑错误，则按照推理逻辑到**第一处错误或中断处**给出相应的局部完成度得分（Partial credit）。
